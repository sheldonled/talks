What's the value of Software?

  • Most people struggle to associate value to intangible things
  • If software is a virtual thing, must we pay for it with real money?!
  • As virtual things are easy to copy and share, we must have ways to protect it.
  • Free Software Movement aims to make sure it isn't true

A matter of freedom, not price

  1. The freedom to run the program as you wish, for any purpose
  2. The freedom to study how the program works, and change it so it does your computing as you wish
  3. The freedom to redistribute copies so you can help your neighbor
  4. The freedom to distribute copies of your modified versions to others

And the revolution started...

GNU → FSF → Linux

Debian → DFSG → Open Source → Eric S. Raymond and Bruce Perens → OSI → OSD

Open Source doesn't simply mean access to the source code

  1. Free Redistribution
  2. Source Code
  3. Derived Works
  4. Integrity of the author's source code
  5. No discrimination against persons or groups
  6. No discrimination against fields of endeavour
  7. Distribution of License
  8. License must not be specific of a product
  9. License must not restrict other software
  10. License must be technology-neutral

Unpleasant things that could happen if the developer has the control

  • Snoop on the user
  • Track the user
  • Restrict the user flow
  • Delete data without consent
  • Compel the user to do something (Like an update)
  • Keep the user unprotected, by restricting the community the possibility of security improvement
  • Sell user data

Free Software is a business like any other

  • You have to know the market
  • Know who your customer is
  • Know what value is added in your product
  • Who are or will be your competitors?
  • Have a business plan
  • Have a marketing plan
  • Have a financial plan
  • Cash flow analysis
